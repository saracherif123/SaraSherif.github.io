Optimizing File Similarity Search Techniques in Data Lakes using Parallel Processing and Big Data Frameworks

Smart cities generate a massive amount of disparate data from multiple sources, necessitating the use of Data Lakes for efficient storage and management, especially with the advent of Big Data. However, it is critical to use appropriate search techniques and governance to prevent the Data Lake from becoming a data swamp. The Data Lake encompasses a wide range of documents and topics, which makes finding similar datasets, e.g., through schema matching approaches, difficult and computationally intensive. In this paper, we propose an approach that leverages Data Lakes and Big Data frameworks to improve the search for related or near-identical structured textual datasets. We present a framework for re-implementing the early pruning algorithm "DS-Prox" within a Big Data framework using Spark. The framework involves prototyping with two machine learning models: a Multilayer Perceptron neural network and a Random Forest ensemble learner. The effectiveness of our approach is demonstrated with good accuracy levels, highlighting the benefits of managing heterogeneous Big Data in Data Lakes.

Sales Dashboard using Tableau

This project aims to develop a detailed and interactive Sales Dashboard that provides comprehensive insights into sales performance, profit, and quantity metrics across different dimensions such as state, segment, and location. The project focuses on analyzing sales, profits, and quantities sold, and comparing this year's KPIs with the previous year's across different states. It also identifies highly competitive salespeople and their contributions to total profits and sales, while determining peak sales and profit points. Key features include displaying total sales, profit, and quantity sold with year-over-year changes, visualizing sales and profit distribution by state, showing monthly sales by segment, and providing a detailed breakdown of sales and profit by segment. Additionally, the dashboard highlights the top 10 states by sales and profit, enabling quick identification of high-performing and underperforming regions.
